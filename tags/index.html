  • “Read Back” uses the Web NFC API (Chrome for Android over HTTPS, user gesture required).[2](https://learn.microsoft.com/en-us/microsoft-edge/extensions/developer-guide/csp)[3](https://developer.chrome.com/docs/privacy-security/permissions-policy)
